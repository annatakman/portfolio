What do I think about Redux?

When we started using Redux a couple of weeks ago I didn't like it at all. It thought it complicated things, and there was so much extra code to write. I use Redux together with React, React is clean, simple and effective. I felt like Redux added so much clutter to my beautiful and clean React code. >>
